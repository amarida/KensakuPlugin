● 概要
　キーワードおよびお天気を検索する「やりますアンコちゃん」
　（以下anko2と表記）のプラグインです。

● 動作条件
　動作しない場合は、.NET Framework の最新バージョンを導入
　してください。

　下記URLからダウンロードしてください。
　http://msdn.microsoft.com/ja-jp/netframework/aa569263.aspx

● インストール
　zipを解凍してできたDLLをpluginsフォルダに置いてください。

　anko2のプラグインに「けんさくさん」が出てきたら完了です。

● アンインストール
　レジストリは触っていません。
　展開したDLLを削除してください。

　KensakuPlugin_for_anko2の設定データは下記のディレクトリに保存されています。
　不要な場合は削除してください。

　XP              ：%USERPROFILE%\Local Settings\Application Data\居酒屋「めがね」\nicoView2.exe_Url_<hash>\<version>\
　VISTA/7/8/8.1/10：%USERPROFILE%\AppData\Local\居酒屋「めがね」\nicoView2.exe_Url_<hash>\<version>\

● 注意事項
　anko2のディレクトリ階層を変更したり、anko2のバージョンを変更すると、
　hash情報が変わるため本プラグインの設定情報は初期化されます。
　設定情報を引き継ぐ場合は、設定ファイルをコピーしてください。
　設定ファイルの場所は前述の「アンインストール」を参照ください。
