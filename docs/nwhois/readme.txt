● 概要
　キーワードを検索するnwhoisプラグインです。

● 動作条件
　動作しない場合は、.NET Framework の最新バージョンを導入
　してください。

　下記URLからダウンロードしてください。
　http://msdn.microsoft.com/ja-jp/netframework/aa569263.aspx

● インストール
　zipを解凍してできたDLLをpluginsフォルダに置いてください。

　nwhoisのプラグインに「けんさくさん for nwhois」が出てきたら完了です。

● アンインストール
　レジストリは触っていません。
　展開したDLLを削除してください。

　KensakuPlugin_for_nwhoisの設定データは下記のディレクトリに保存されています。
　不要な場合は削除してください。

　XP     ：%USERPROFILE%\Local Settings\Application Data\nwhois\nwhois.exe_Url_<hash>\1.0.0.0\
　VISTA/7：%USERPROFILE%\AppData\Local\nwhois\nwhois.exe_Url_<hash>\1.0.0.0\

● 注意事項
　nwhoisのディレクトリ階層を変更すると、hash情報が変わるため
　本プラグインの設定情報は初期化されます。
　設定情報を引き継ぐ場合は、設定ファイルをコピーしてください。
　設定ファイルの場所は前述の「アンインストール」を参照ください。
